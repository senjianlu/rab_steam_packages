# rab_steam_packages

## 项目介绍  
基于 [steampy](https://github.com/bukson/steampy) 的改进后版本，对功能和逻辑仅进行少量更改，项目成立之初主要的工作为补全注释以提供便利，后续为随 Steam 接口版本迭代适配更新。  

## 环境
| 系统 | 版本 |  
| -----| ---- |  
| Linux | CentOS7 |  

| 语言 | 版本 |
| -----| ---- |  
| Python | 3.8.2 | 

## 使用方法  
依赖此仓库:
```bash
git submodule add https://github.com/senjianlu/rab_python_packages.git
```
